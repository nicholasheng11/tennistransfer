How to export this as the social-preview image

  1. This page must be opened locally — not linked from anywhere on the live site.
  2. Open this file in Chrome or Edge: right-click anywhere in the navy card below → Inspect.
  3. In the Elements panel, locate the <div class="og"> element. Right-click it → Capture node screenshot.
  4. Chrome saves a PNG at the element's exact size (1200×630). Move/rename it to og-image.png in this folder, next to index.html.
  5. Tell me when it's in the repo and I'll update the og:image / twitter:image meta tags in index.html to point at it.

If you'd rather use a different tool: any 1200×630 PNG with this layout works. The dimensions are what matter for Twitter/X, iMessage, Slack, etc. to render a large preview card.
